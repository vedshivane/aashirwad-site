# Context

This folder holds non-production material that was previously scattered across the repo:

- `agents/`: local design and workflow reference material
- `brand/`: original logo source files kept for asset regeneration
- `images/`: input/reference images supplied during development
- `notes/`: transcripts, planning notes, and scratch documentation
- `scripts/`: one-off asset generation and debug utilities

The deployable app continues to live in the repo root under `app`, `components`, `lib`, and `public`.
